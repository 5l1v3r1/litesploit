## Installation

#### Requirements
* PHP 5 or higher
* GCC
* Zephir lang
* PHP-Dev
* libpcre3-dev

For Linux Ubuntu and Debian
```
sudo apt-get install -y gcc make re2c php5-cli php5-dev php5-curl php5-json libpcre3-dev
```

#### Ubuntu / Debian
```bash
git clone https://github.com/devilscream/litesploit.git
cd litesploit
./install
```
